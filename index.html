The N2O Ecosystem

Introduction

N2O defines a way to scale applications from protocols to modules, events persistence and schema. It replaces CORBA/SOAP/XMPP in small and compact binary manner, yet it's ready for high-speed, low-latency IoT, MQTT, and WebSocket applications. You can embed it into cowboy, emqttd or any message processing loop.
